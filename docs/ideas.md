# アプリアイディア集

## 実用系

### ポモドーロタイマー
- **難易度**: 低
- **コンテナ**: テキスト x1
- **概要**: 作業25分/休憩5分のタイマー。残り時間とUnicodeプログレスバーを表示。クリックで開始/停止、ダブルクリックでリセット
- **状態**: 開発中

### 天気＋予定グランス
- **難易度**: 中
- **コンテナ**: テキスト x2（天気 + 予定）
- **概要**: 天気APIとGoogleカレンダーから情報を取得し、1画面に表示。バックエンドでAPI処理。定期的に自動更新
- **バックエンド**: 必要（API key管理）

### 語学フラッシュカード
- **難易度**: 低〜中
- **コンテナ**: テキスト x1
- **概要**: 表面（単語）→クリックで裏面（意味）→スクロールで次のカード。Anki的な間隔反復も可能。データはlocalStorageかバックエンド
- **バリエーション**: 英単語、漢字、プログラミング用語など

### 通知ビューア
- **難易度**: 中〜高
- **コンテナ**: リスト x1 + テキスト x1
- **概要**: Slack/GitHub等の未読通知をリストで一覧表示、クリックで詳細をテキスト表示。バックエンドでAPI連携
- **バックエンド**: 必要（OAuth, webhook）

## ゲーム系

### 数当てゲーム (Higher/Lower)
- **難易度**: 低
- **コンテナ**: テキスト x1
- **概要**: 1-100のランダムな数を当てる。スクロールで予想を上下、クリックで回答。「もっと大きい/小さい」のフィードバック

### テキストアドベンチャー
- **難易度**: 中
- **コンテナ**: テキスト x1 + リスト x1
- **概要**: 選択肢付きストーリー。テキストコンテナで物語を表示、リストコンテナで選択肢。ページフリップで長文対応
- **発展**: LLMでストーリー動的生成

### リアクションゲーム
- **難易度**: 低
- **コンテナ**: テキスト x1
- **概要**: ランダムなタイミングで「TAP!」と表示、できるだけ速くクリック。反応速度を計測・記録

## 実験系

### AIアシスタント
- **難易度**: 高
- **コンテナ**: テキスト x1
- **概要**: マイクで音声入力→バックエンドでWhisper+LLM処理→テキストで回答表示。ハンズフリーAI体験
- **バックエンド**: 必要（Whisper, Claude API）
- **注意**: PCM 16kHz S16LE mono、マイクは`audioControl`で制御

### 音声メモ
- **難易度**: 中
- **コンテナ**: テキスト x1 + リスト x1
- **概要**: マイクで録音→Whisperで文字起こし→テキスト表示。メモ一覧をリスト表示
- **バックエンド**: 必要（Whisper API）
